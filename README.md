## ✨ 核心亮点

### 🚀 静默自动化
无需任何点击！当链接在浏览器中打开为“长按复制”提示页或安全中转页时，插件会自动检测并瞬间跳转到原始网页。

### 🔮 AI 情报卡片 (New!)
点击插件图标，即可展开强大的**情报中心**。
- **链接去毒**：在弹窗手动打开流程中，可选移除 `utm_*`, `fbclid` 等常见追踪参数，保护你的隐私。
- **智能摘要**：(Beta) 快速分析当前页面内容，生成核心观点摘要，提升阅读效率。

### 🛡️ 隐私与安全
- **零数据上传**：所有链接解析完全在本地完成，绝不上传任何用户数据。
- **权限最小化**：仅申请必要的标签页和特定域名权限，不滥用剪贴板或全网访问权。

### 🎨 极简美学
- **状态感知 UI**：点击插件图标，直观展示“保护运行中”状态。
- **玻璃拟态设计**：精美的磨砂玻璃视觉效果，与现代操作系统完美融合。
- **兜底支持**：提供极简的手动输入框，应对极少数无法自动识别的复杂场景。

## 🛠️ 功能特性

1.  **自动直达**：智能识别 `weixin110.qq.com`、`mp.weixin.qq.com` 等域名的中转/拦截页，自动提取 URL 参数、脚本变量或 DOM 文本中的真实链接并跳转。
2.  **链接去毒**：在 Popup 面板中开启“链接去毒”后，弹窗手动打开的链接会在本地去除常见追踪参数。
3.  **智能辅助**：内置 AI 深度分析，可优先使用浏览器内置本地引擎；本地不可用时可使用你配置的云端 API。
4.  **手动模式**：点击插件图标，可手动粘贴被拦截的链接，一键还原并打开。

## 📦 安装说明

1.  打开 Chrome/Edge 浏览器的“扩展程序”管理页面 (`chrome://extensions/` 或 `edge://extensions/`)。
2.  开启“开发者模式”。
3.  选择“加载已解压的扩展程序”，选中本项目的 [extension](extension/) 目录。

## 📖 使用指南

- **日常使用**：保持插件开启即可。当你在微信 PC 版点击链接唤起浏览器时，插件会自动工作。
- **进阶功能**：点击插件图标，点击“🔮 AI 深度分析”按钮，体验更多高级功能。

## 🔒 隐私政策

本插件致力于保护用户隐私：
- 不收集、存储或发送任何用户浏览记录或链接数据。
- 不读取剪贴板内容（除非用户在弹窗中手动粘贴）。
- 核心跳转与去毒逻辑在本地运行；AI 分析仅在你主动触发时运行，且在本地不可用时才会使用你配置的第三方接口。
- 详细隐私声明请见 [PRIVACY_POLICY.md](PRIVACY_POLICY.md)。
